# В первую очередь бота нужно сделать администратором беседы. Без этого бот работать не будет.
# Далее беседу надо зарегистрировать. Делается это командой
# /reg on
# В ответ бот сообщит ваш личный регистрационный номер. Он нужен, чтобы давать команды боту в личных сообщениях.  После этого ботом можно пользоваться.
# Команды доступны только владельцу беседы. Исключение составляет команда /post, которая доступна всем участникам беседы.
# Командами /reg можно пользоваться только в беседе. Остальными командами можно пользоваться и в беседе и в личной переписке с ботом.
#
# Команды
#
#
# Узнать свой регистрационный номер:
# /reg info
# Отменить регистрацию (бот работать не будет):
# /reg off
#
# Зарегистрировать группу для рандомных постов (подставить ссылку на нужную группу) и разрешить команду /post:
# /grouppost https://vk.com/link_to_the_group
# Получить инфу по группе для рандомных постов:
# /grouppost info
# Запретить команду /post (группа для рандомных постов будет удалена):
# /grouppost off
# Отправить рандомный пост со стены выбранной для этого группы:
# /post
#
# Зарегистрировать группу, в которой должны состоять участники беседы (подставить ссылку на нужную группу) и разрешить команду /purge:
# /grouppurge https://vk.com/link_to_the_group
# Получить инфу по группе, зарегистрированной командой /grouppurge:
# /grouppost info
# Запретить команду /purge (группа, зарегистрированная командой /grouppurge, будет удалена):
# /grouppost off
# Удалить из беседы людей, которые не состоят в группе, зарегистрированной командой /grouppurge :
# /purge
#
# Бот позволяет включить режим "интервал". В этом режиме каждые -n сообщений бот отправляет рандомную фразу из предварительно сохраненных.
# Зарегистрировать фразы, которые будут использоваться в режиме "интервал":
# /phrase add Фраза первая | Фраза вторая | Фраза третья | Фраза четвертая
# Фразы можно регистрировать по одной, а можно сразу несколько. Вертикальная черта является разделителем и не может присутствовать во фразе.
# Прислать все зарегистрированные фразы и их айдишники:
# /phrase info
#
# Удалить фразы:
# /phrase remove 1 2 3
# Для удаления фраз нужно указать их айдишники через пробел.
#
# Удалить все фразы и отключить режим "интервал":
# /phrase remove all
#
# Режим "интервал"
#
# Включить режим "интервал" и установить число сообщений, через которое бот будет отправлять одну из сохраненных фраз:
# /interval n
# где n - число от двух до 1000
# Выключить режим интервал:
# /interval off
# Получить информацию по режиму интервал:
# /interval info
#
# Все команды, кроме команд /reg, доступны для владельца беседы в личной переписке с ботом. Чтобы воспользоваться командой, нужно вторым словом в сообщении указать регистрационный номер чата, например:
# /post 2000000001
# /interval 2000000001 info
# /phrase 2000000001 remove 1 2 3
# /phrase 2000000001 add Фраза первая | Фраза вторая
