2. Далее нужно, чтобы бот зарегистрировал вашу беседу. Для этого в чате пишем команду

В ответ бот сообщит ID беседы. Этот номер нужен, чтобы давать команды боту в личных сообщениях.
После регистрации ботом можно пользоваться.
3.Команды
Команды доступны только владельцу беседы. Исключение составляет команда /post, которая доступна всем участникам беседы.
1) /reg - регистрация.
Командами /reg можно пользоваться только в беседе. Остальными командами можно пользоваться и в беседе и в личной переписке с ботом.
1. Узнать ID чата:
/reg info
2. Отменить регистрацию (бот работать не будет):
/reg off
3. Зарегистрировать чат (после отмены регистрации тоже):
/reg on
Все команды, кроме команд /reg, доступны для владельца беседы в личной переписке с ботом.
Чтобы воспользоваться командой в личке, нужно вторым словом в сообщении указать ID чата, например:
/post 200000003
/kick 200000003 on
/newpost 200000003 group https://vk.com/link_to_the_group
/phrase 200000003 add Первая фраза | Вторая фраза | Третья фраза
2) /post - рандомные посты.
По команде /post бот пришлет в чат случайный пост со стены выбранной вами группы. Группа при этом не должна быть частной или закрытой.
1. Выбрать группу:
/post group https://vk.com/link_to_the_group
После команды /post group нужно подставить ссылку на группу или её ID или screen_name(короткое имя). Если группа с таким адресом не существует или является закрытой, бот об этом сообщит.
Поменять уже выбранную группу можно этой же командой.
2. Включить команду /post:
/post on
После того как вы выбрали группу и сохранили её в настройках, нужно дать боту разрешение отправлять в чат случайные посты по команде /post.
3. Попросить бота прислать какой-нибудь пост со стены выбранной вами группы:
/post
Этой командой могут пользоваться все участники беседы, не только её владелец.
4. Запретить команду /post:
/post off
Сохраненная группа останется в настройках, но команду /post бот будет игнорировать. Разрешить боту снова отправлять случайные посты можно командой /post on
5. Удалить из настроек группу для команды /post:
/post group delete
Команда /post будет отключена.
6. Посмотреть свои настройки для команды /post:
/post info
Бот пришлет информацию о сохраненной группе для команды /post.
3) /kick - удалять участников беседы.
По команде /kick бот удалит участников беседы, которые не являются участниками выбранной вами группы.
Группа при этом не должна быть частной или закрытой. Бот не удаляет из группы админов и других ботов.
Только владелец беседы может пользоваться этой командой.
1. Выбрать группу:
/kick group https://vk.com/link_to_the_group
После команды /kick group нужно подставить ссылку на группу или её ID или screen_name(короткое имя). Если группа с таким адресом не существует или является закрытой, бот об этом сообщит.
Поменять уже выбранную группу можно этой же командой.
2. Включить команду /kick:
/kick on
После того как вы выбрали группу и сохранили её в настройках, нужно дать боту разрешение удалять участников чата по команде /kick.
3. Удалить из беседы людей, которые не состоят в выбранной вами группе:
/kick
4. Отключить команду /kick:
/kick off
Сохраненная группа останется в настройках, но команду /kick бот будет игнорировать(не будет удалять участников беседы). Разрешить боту снова удалять людей из чата можно командой /kick on
5. Удалить из настроек группу для команды /kick:
/kick group delete
Команда /kick будет отключена.
6. Посмотреть свои настройки для команды /kick:
/kick info
Бот пришлет информацию о сохраненной группе для команды /kick.
4)/newpost - получать посты выбранной группы по мере их появления на стене.
Если включить этот режим, бот будет отправлять в чат новый пост, как только он появился на стене выбранной группы.
1. Выбрать группу:
/newpost group https://vk.com/link_to_the_group
После команды /newpost group нужно подставить ссылку на группу или её ID или screen_name(короткое имя). Если группа с таким адресом не существует или является закрытой, бот об этом сообщит.
Поменять уже выбранную группу можно этой же командой.
2. Включить режим /newpost:
/newpost on
После того как вы выбрали группу и сохранили её в настройках, нужно дать боту разрешение присылать с её стены новые посты.
3. Отключить режим /newpost:
/newpost off
Сохраненная группа останется в настройках, но обновления присылаться не будут. Разрешить боту снова присылать посты можно командой /newpost on
4. Удалить из настроек группу для команды /newpost:
/newpost group delete
Команда /newpost будет отключена.
5. Посмотреть свои настройки для команды /newpost:
/newpost info
Бот пришлет информацию о сохраненной группе для команды /newpost.
5) /interval - получать сообщения от бота через определенное число сообщений других участников беседы.
В режиме "интервал" бот будет присылать случайные сообщения в чат. Сообщения нужно предварительно сохранить в настройках с помощью команды /phrase (см.ниже).
1. Установить интервал между сообщениями бота:
/interval 5
Бот будет отправлять сообщения через определенный интервал, т.е. после некоторого количества сообщений других участников чата.
Установить интервал можно командой /interval, указав после нее нужное количество сообщений, например /interval 5.
Это значит, что бот будет присылать случайную фразу через каждые 5 сообщений других участников чата.
2. Включить режим /interval:
/interval on
После того как вы выбрали интервал и сохранил его в настройках, нужно дать боту разрешение присылать случайные фразы.
Если у вас не сохранено ни одной фразы, режим интервал работать не будет. Как сохранить фразы, см. ниже.
3. Отключить режим /interval:
/interval off
Бот перестанет присылать случайные фразы.
4. Посмотреть свои настройки для команды /interval:
/interval info
6) /phrase - управлять фразами для режима /interval
1. Сохранить фразу:
/phrase add Первая фраза | Вторая фраза | Третья фраза
Командой /phrase add можно сохранять как одну фразу, так и несколько - в этом случае разделяйте фразы вертикальными чертами, как в примере выше.
2. Получить информацию о сохраненных фразах:
/phrase info
Бот пришлёт все ваши сохраненные фразы и их ID.
3. Удалить фразу:
/phrase remove ID
Если вы больше не хотите, чтобы какую-либо из ваших фраз бот присылал в режиме /interval, нужно написать команду /phrase remove и указать ID этой фразы.
ID фразы можно узнать командой /phrase info.
4. Удалить все фразы:
/phrase remove all
Все сохраненные фразы удалятся, режим /interval будет отключен.
6) /smart - бот будет отвечать на определенные фразы. Фразы-триггеры и ответы на них можно сохранить командой /smartreply.
1. Включить режим /smart:
/smart on
Включить режим можно, только если вы у вас есть сохраненные smart-сообщения (см. команду /smartreply).
2. Отключить режим /smart:
/smart off
3. Узнать, включен режим /smart или выключен:
/smart info
7) /smartreply - управлять smart-сообщениями для режима /smart
1. Сохранить smart-сообщение:
/smartreply add Входящее сообщение || Ответ бота
После команды /smartreply add пишем сообщение, на которое будет реагировать бот, ставим разделитель - две вертикальные черты, далее пишем ответ, который будет отправлять бот.
Максимальная длина сообщения-триггера - 100 знаков.
Максимальная длина сообщения-ответа - 100 знаков.
2. Удалить smart-сообщение:
/smartreply remove
Если вы больше не хотите, чтобы бот отвечал на какое-либо входящее сообщение, нужно написать команду /smartreply remove и указать ID этого smart-сообщения.
ID smart-сообщения можно узнать командой /smartreply info.
3. Удалить все smart-сообщения:
/smartreply remove all
Все сохраненные smart-сообщения удалятся, режим /smart будет отключен.
4. Получить информацию о сохраненных smart-сообщенияx:
/smartreply info
Бот пришлёт все ваши сохраненные smart-сообщенияx и их ID.
